What I Need This For
In TinyCommand, the embed integration works like this:

Host app creates an iframe pointing to {STUDIO_URL}/embed?type=table
Iframe loads and sends a ready postMessage event to the host
Host sends setAuth with authentication tokens and server config
Iframe confirms with authConfigured
Host sends setTheme with visual customization
Host sends setMode with "edit" | "preview" | "readonly"
Host sends loadAsset with the table's configuration data (column definitions, view settings, etc.)
Iframe renders the table and confirms with assetLoaded
User edits in the iframe trigger assetUpdated events back to the host with the modified configuration
Host debounces and persists updates via its own API
